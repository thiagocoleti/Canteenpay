# Documento de descrição do projeto

Este documento descreve o projeto que deverá ser criado por uma IA Generativa para resolver um determinado problema.

# Contexto
A cantina da escola pediu para criar uma solução para registrar os gastos dos alunos durante o intervalo. No final do mês, o aplicativo deve enviar para os pais o valor para pagamento. 

# Problema
O principal problema é comunicar com os pais no final do mês e garantir o pagamento porque não podemos impedir os alunos de comprar

# Papel da IA o
Comporte-se como um design.

# O que a IA deve fazer
 Proponha uma solução computacional - pode ser um aplicativo ou website.

# O que a IA deve entregar
Me devolva uma documentação escrita descrevendo o projeto, e um protótipo escrito em HTML, CSS e JavaScript, organizado conforme melhores padrões de codificação. O protótipo deve estar organizado em um pasta dentro de um arquivo winRAR para eu baixar e testar.

# Melhorias para a segunda versão
Na interface inicial deve ter uma tabela, na parte inferior contendo os registros de vendas diários e mensais.
